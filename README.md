## Usage

Download eslint-config-coveo with `npm`

```sh
  npm install --save-dev eslint-config-coveo
```

Extends Coveo's rules in your .eslintrc files

In `YAML`

```yaml
---
  extends: coveo
  rules:
    ...
```

In `JSON`

```json
{
  "extends": "coveo",
  "rules": {

  }
}
```
